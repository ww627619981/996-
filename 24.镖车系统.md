## 接口

### 镖车自动寻路到指定坐标
`dartmap`

|参数|类型|空|默认|注释|
|:----    |:-------    |:--- |----|------      |
|play     |`object`      |否   |    |   玩家对象 |
|aimX     |`integer`      |否   |    |   目标X坐标 |
|aimY     |`integer`      |否   |    |   目标Y坐标 |
|range     |`integer`      |否   |    |  人物离镖车距离内自动寻路<br />取值范围：0-12<br />0-不检测  |

### 人物下线，镖车存活设置

`darttime`

|参数|类型|空|默认|注释|
|:----    |:-------    |:--- |----|------      |
|play     |`object`      |否   |    |   玩家对象 |
|time     |`integer`      |否   |    |   镖车存活时间，秒 |
|isDie     |`integer`      |否   |    |   下线是否消失<br />0-消失，1-时间到达消失 |
